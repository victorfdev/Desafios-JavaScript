# 1° dia - Tamanho da String

## **Dificuldade**: ⭐☆☆☆☆

### **Descrição do Problema**

Dada uma String qualquer, imprima na tela a quantidade de caracteres que essa String possui

### **Exemplo de Entrada:**

```js
"Olá, tudo bem?"
```

### **Exemplo de Saída:**

```js
14
```
